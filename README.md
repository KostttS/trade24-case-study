# Trade24 — Classifieds Marketplace Mobile App

Mobile classifieds marketplace app for publishing, browsing and managing listings across different categories.

The app was built around a simple marketplace flow: users can browse listings, search by category, publish their own offers and receive app notifications about listing activity.

## Technical challenges

### Listing publishing flow

The app required a clear flow for creating and publishing listings, including category selection, listing data, images and status updates.

### Marketplace data structure

The project needed a practical Firestore structure for users, listings, categories, images and listing statuses.

### User notifications

The app included notification-style screens to inform users when a listing was published, updated or received activity.

### Mobile-first UX

The main challenge was to keep the classifieds experience simple: users should quickly find listings or publish their own offer without a complicated interface.
